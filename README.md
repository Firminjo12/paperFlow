# 🚀 paperFlow - La Suite PDF Ultime

paperFlow est une plateforme web moderne et puissante permettant de gérer, convertir et éditer vos fichiers PDF en toute simplicité. Inspirée des meilleurs outils du marché, elle offre une expérience utilisateur fluide et premium.

## ✨ Fonctionnalités principales

paperFlow regroupe tous les outils essentiels pour vos documents :

### 🔓 Sécurité & Organisation
*   **Unlock PDF** : Supprimez les mots de passe et les restrictions des PDF protégés.
*   **Protect PDF** : Sécurisez vos documents avec un cryptage robuste.
*   **Merge & Split** : Fusionnez plusieurs fichiers ou divisez un PDF en plusieurs documents.
*   **Organize** : Réorganisez, supprimez ou faites pivoter les pages de vos PDF.

### 🔄 Conversion Haute Qualité
*   **Office vers PDF** : Convertissez Word, Excel et PowerPoint vers PDF.
*   **PDF vers Office** : Transformez vos PDF en fichiers DOCX, XLSX ou PPTX éditables.
*   **Images vers PDF** : Créez des PDF à partir de vos photos (JPG, PNG).

### 🛠️ Édition Avancée
*   **Éditeur PDF** : Ajoutez du texte, des images ou dessinez directement sur vos documents.
*   **Signature électronique** : Signez vos documents de manière numérique.
*   **OCR** : Extrayez le texte d'images ou de scans PDF.
*   **Watermark** : Ajoutez des filigranes personnalisés pour protéger vos droits.

## 🛠️ Stack Technique

*   **Frontend** : [React](https://reactjs.org/) + [Vite](https://vitejs.dev/) + [Vanilla CSS](https://developer.mozilla.org/en-US/docs/Web/CSS)
*   **Backend** : [Node.js](https://nodejs.org/) + [Express](https://expressjs.com/)
*   **Base de données** : [MongoDB](https://www.mongodb.com/)
*   **Traitement PDF** : [qpdf](http://qpdf.sourceforge.net/), [libreoffice-convert](https://www.npmjs.com/package/libreoffice-convert), [pdf-lib](https://pdf-lib.js.org/)
*   **Conteneurisation** : [Docker](https://www.docker.com/) & Docker Compose

## 🚀 Installation Rapide

La méthode la plus simple pour lancer paperFlow localement est d'utiliser **Docker**.

### Pré-requis
*   Docker & Docker Compose installés sur votre machine.

### Étapes
1.  **Cloner le dépôt** :
    ```bash
    git clone https://github.com/Firminjo12/SingnFlow.git
    cd SingnFlow
    ```

2.  **Lancer l'application** :
    ```bash
    docker-compose up --build
    ```

3.  **Accéder à l'application** :
    *   Frontend : `http://localhost:5173`
    *   Backend : `http://localhost:5000`
